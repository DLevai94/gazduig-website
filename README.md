# Gaxduig Website

## Get started

Install the Gatsby CLI:

```sh
npm i -g gatsby-cli
```

Clone repo

```sh
gcl <repo-url>
```

Yarn install & run:

```sh
cd gazduig-website
yarn
yarn develop
```

## Format and lint

- `yarn analyze` - See what ESLint and Prettier can fix
- `yarn fix` - Run Prettier and ESLint with the `--fix` option

## Build

Use `yarn build` to build for production.
